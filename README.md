# 🔗 Shad Link Extractor (ابزار استخراج لینک شاد)

ابزاری برای **Termux** که لینک‌های پنهان برنامه شاد را ضبط و ذخیره می‌کند.

---

## 📋 پیش‌نیازها

- **Termux** نصب‌شده از [F-Droid](https://f-droid.org/packages/com.termux/)
- اندروید ۷ یا بالاتر
- اتصال WiFi فعال

---

## 🚀 نصب سریع

مراحل زیر را **یک‌بار** در Termux انجام دهید:

```bash
# ۱. کلون کردن مخزن
git clone https://github.com/bc95game-art/Nnnnn.git shad-extractor
cd shad-extractor

# ۲. نصب خودکار
bash install.sh
```

---

## 📡 روش ۱ — ضبط ترافیک (پیشنهادی)

### مرحله ۱: راه‌اندازی پروکسی

```bash
cd shad-extractor
bash run.sh
```

### مرحله ۲: تنظیم WiFi گوشی

۱. وارد **تنظیمات WiFi** شوید  
۲. روی شبکه‌ی متصل **نگه دارید** ← «تغییر شبکه» یا «پیشرفته»  
۳. «پروکسی» را روی **دستی (Manual)** بگذارید  
   - **آدرس:** `127.0.0.1`  
   - **پورت:** `8080`  

### مرحله ۳: نصب گواهی SSL

```bash
bash install_cert.sh
```

سپس دستورالعمل روی صفحه را دنبال کنید (نصب گواهی mitmproxy در اندروید).

### مرحله ۴: باز کردن شاد

برنامه شاد را باز کنید. لینک‌ها به‌صورت خودکار ضبط می‌شوند.

### مرحله ۵: خروجی گرفتن

```bash
bash export_links.sh
```

فایل لینک‌ها در **دانلودهای گوشی** ذخیره می‌شود.

---

## 📋 روش ۲ — ابزار تعاملی (Interactive)

```bash
bash interactive.sh
```

این حالت:
- لینک‌های Clipboard را تجزیه می‌کند
- URL های وارد شده را آنالیز می‌کند
- لینک‌های پنهان (deep link) را نمایش می‌دهد

---

## 📁 فایل‌های خروجی

| فایل | محل |
|------|-----|
| لینک‌های خام | `~/storage/downloads/shad_links_تاریخ.txt` |
| لاگ کامل | `~/shad-extractor/logs/captured.log` |

---

## ⚠️ نکات مهم

- بعد از اتمام کار، پروکسی WiFi را **غیرفعال** کنید
- این ابزار فقط برای **استفاده شخصی** است
- در بعضی گوشی‌ها ممکن است نیاز به **روت** باشد (برای SSL Pinning)

---

## 🆘 عیب‌یابی

| مشکل | راه‌حل |
|-------|--------|
| اینترنت قطع شد | پروکسی WiFi را غیرفعال کنید |
| شاد باز نمی‌شود | گواهی SSL را نصب کنید (`bash install_cert.sh`) |
| لینکی ضبط نشد | مطمئن شوید پروکسی درست تنظیم شده |
